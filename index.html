REFLECT: Summarizing Robot Experiences for FaiLure Explanation and CorrecTion

The ability to analyze failed executions and self-recover from failure is crucial for an explainable and robust robotics system. Large Language Models (LLMs) have recently demonstrated strong common sense reasoning skills on textual inputs. To leverage the power of LLM for robot failure analysis, we propose a framework REFLECT, which converts multi-sensory data into a hierarchical summary of robot past experiences and queries LLM with a progressive failure analysis algorithm. Finally, conditioned on the explanation, a language planner generates a feasible plan for the robot to recover from failure and complete the task. To systematically evaluate the framework, we create the RoboFail dataset and show that our method is able to generate informative failure explanations that can assist in successful correction planning. The dataset and code will be publicly available.
